A collection of **C programs demonstrating Linux file handling using low-level system calls**.

These programs help understand how the **Linux kernel interacts with files using file descriptors and system calls** such as:

* `creat()`
* `open()`
* `read()`
* `write()`
* `close()`

This repository is useful for learning **Linux System Programming**, **Operating System concepts**, and **low-level file manipulation in C**.

---

# 📚 Concepts Covered

The programs demonstrate important Linux file system operations.

## File System Operations

* Creating files
* Opening files
* Reading file data
* Writing data to files
* Closing files

## System Programming Concepts

* File Descriptors
* File Permissions (`0777`)
* Buffer Handling
* User Input Handling
* Basic Error Handling

## Linux System Calls

| System Call | Purpose               |
| ----------- | --------------------- |
| `creat()`   | Create a new file     |
| `open()`    | Open an existing file |
| `read()`    | Read data from a file |
| `write()`   | Write data to a file  |
| `close()`   | Close an opened file  |
